# Module 5: Harness Sign-Off And Adoption Plan

Time: 25 minutes

## Purpose

Complete [`docs/research_article_harness.md`](../docs/research_article_harness.md)
and write a 7-day plan tied to checklist rows.

## Harness sign-off

Open the checklist in `docs/research_article_harness.md`. Mark each row you can
demonstrate from your `starter_article` copy (or equivalent).

Update `notes/orchestration_log.md` with what you delegated vs reviewed yourself.

## Adoption plan

Create `outputs/adoption_plan.md` from
[`templates/adoption_plan.md`](../templates/adoption_plan.md).

Prompt:

```text
Read docs/research_article_harness.md and my outputs/adoption_plan.md draft.
Suggest one smaller first task if my plan is too broad.
```

## Pair check

1. Is the first task bounded?
2. Are data boundaries explicit?
3. Is verification concrete?
4. Does the plan name which claims stay human-only?
